TaskFlow To-Do List Mobile Application

Project Description
TaskFlow is a mobile To-Do List application developed using Unity and C#. The application helps users manage their daily tasks in a simple and interactive way. Users can create an account, log in, add tasks, mark them as completed, and track their progress. The app provides a more engaging and efficient alternative to traditional task management methods such as paper notes.

Technologies Used
Unity Version: Unity 2023 LTS (Latest Stable Version)
Programming Language: C#
iOS SDK: iOS 17 SDK (via Xcode latest version)
Firebase SDK: Firebase Unity SDK (Authentication & Database)
Database: SQLite (local storage)
Data Format: JSON
Main Features
User Sign Up & Login (Authentication)
Add new tasks
View task list
Mark tasks as completed
Delete tasks
Track task progress (completed vs remaining)
How to Run the Project
Install Unity Hub and Unity 2023 LTS
Open the project using Unity Hub
Make sure iOS Build Support is installed
Open the project scene (Main/Dashboard)
Connect Firebase (if required)
Click Play to run in Unity Editor
For iOS Build:
Open Build Settings in Unity
Select iOS Platform
Click Build
Open the project in Xcode
Run on iPhone or simulator
Third-Party Plugins Used
Firebase Unity SDK
SQLite for Unity

Project Idea 
The main idea of this project is to create a simple and user-friendly task management application that helps users organize their daily activities. The application also includes progress tracking to motivate users and improve productivity.
